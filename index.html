Our products

Touch Track

Touch Track is an issue reporting, beta test management platform, built for the Android platform. What sets Touch Track apart is capturing the context leading up to crashes and issues.

OrmLite (Contributor)

Object Relational Mapping Lite (ORM Lite) provides some simple, lightweight functionality for persisting Java objects to SQL databases while avoiding the complexity and overhead of more standard ORM packages.

Touch Lab Draw! (Coming Soon)

A little experiment app that lets you draw a picture for our website right from your phone. Download, log in (with twitter), and draw a picture. It'll show up on our home page.

Superbus (Creator)

Superbus is a synchronization bus service for Android. The bus keeps commands in order, processes them one at a time, and will persist the command to be run later if the system can’t handle the command now.

HTMLSelect (Coming Soon)

Android doesn't provide a way to step into the text select process. HTMLSelect allows touch-based selection of text using Javascript.
